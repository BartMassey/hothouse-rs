# hothouse-rs: Cleveland Music Hothouse pedal support crate
Bart Massey 2026

This Rust library crate sits atop the
[`daisy`](https://crates.io/crates/daisy) crate to provide
suport for the Cleveland Music Hothouse pedal. The Hothouse
is a pedal that wraps the Electrosmith Daisy Seed processor
board.

## Acknowledgments

Much of the code here was taken from the `daisy` crate
examples.
